### Instalation

Example:

```sh
wget https://github.com/gokcehan/lf/releases/download/r25/lf-linux-amd64.tar.gz
tar zxf lf-linux-amd64.tar.gz \
&& rm lf-linux-amd64.tar.gz \
&& mkdir -p ~/.local/bin \
&& mv ./lf ~/.local/bin/
```
